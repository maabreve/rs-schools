# Escolas RS

## Links

## Technologies

- React

## Installation

- git clone https://github.com/maabreve/rs-schools
- cd rs-schools
- yarn


## Api Configuration
- .env file - Change REACT_APP_API_URL


## Development Running

- yarn start

## Build

- yarn build
